一、评估指标（★ 最高频，几乎必考）
================================================================================

混淆矩阵速记：

              预测正例    预测负例
    真实正例     TP         FN
    真实负例     FP         TN

  Accuracy  = (TP+TN) / (TP+TN+FP+FN)      总体准确率
  Precision = TP / (TP+FP)                  预测为正的中有多少真阳性
  Recall    = TP / (TP+FN)                  真实为正的中找出了多少
  F1        = 2·P·R / (P+R)                 调和平均

  TPR = Recall（真阳性率）
  FPR = FP / (FP+TN)                        （假阳性率）

1. AUC（必背！★★★★★）
--------------------------------------------------------------------------------
  物理含义：随机抽取一个正例和一个负例，正例得分高于负例的概率

    AUC = P(S_正 > S_负)

  几何含义：ROC曲线下的面积，值域[0, 1]，0.5 = 随机

  手算（排序法）：
    ① 按得分从高到低排序
    ② 统计正例的 rank（排名），设总和为 sum_pos_rank
    ③ n_pos = 正例数，n_neg = 负例数
    ④ AUC = (sum_pos_rank - n_pos·(n_pos+1)/2) / (n_pos × n_neg)

  快速判断：
    AUC ≈ 0.9 → 极好
    AUC ≈ 0.7 → 一般
    AUC = 0.5 → 等于瞎猜

2. 多分类加权指标（荣耀2024/2025真题）★★★★★
--------------------------------------------------------------------------------
  加权精确率 = Σ (类别i样本数/总样本数 × 类别i精确率)
  加权召回率 = Σ (类别i样本数/总样本数 × 类别i召回率)
  加权F1     = 2 × 加权P × 加权R / (加权P + 加权R)

  按各类别真实样本数占比作为权重。

3. ROC vs PR 曲线选择
--------------------------------------------------------------------------------
  PR曲线（Precision-Recall）：用于正例稀少（imbalance严重）时
  ROC曲线：用于类别较平衡时

二、线性回归与正则化（★ 高频）
================================================================================

4. 线性回归解析解（最小二乘法）
--------------------------------------------------------------------------------
  损失函数：L(w) = ||Xw - y||²

  解析解：w* = (XᵀX)⁻¹ Xᵀy

  前提：XᵀX 必须可逆（满秩），否则需用伪逆或正则化

5. 岭回归（Ridge L2正则化）★★★★★
--------------------------------------------------------------------------------
  损失函数：L(w) = ||Xw - y||² + λ||w||²

  解析解：w* = (XᵀX + λI)⁻¹ Xᵀy

  作用：λ↑ → w趋向0 → 防止过拟合
       解决 XᵀX 病态（不满秩）问题

  对比：
    L1正则化（Ridge）→ 使w稀疏（特征选择），解在顶点
    L2正则化（Lasso）→ 使w平滑，趋向0但不为0

6. 过拟合 vs 欠拟合
--------------------------------------------------------------------------------
  欠拟合：训练、测试误差都高 → 加特征 / 提高模型复杂度
  过拟合：训练误差低、测试误差高 → 正则化 / 加数据 / 减少特征

  偏差-方差分解：
    E_test = Bias² + Variance + Irreducible Noise
    欠拟合 → 高偏差；过拟合 → 高方差

三、分类模型基础（★ 高频）
================================================================================

7. 逻辑回归（Logistic Regression）
--------------------------------------------------------------------------------
  sigmoid:  σ(z) = 1 / (1 + e⁻ᶻ)

  预测：P(y=1|x) = σ(w·x)，>0.5 → 正例
  损失（对数损失）：L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]

  优点：可解释性强（odds ratio），输出概率，训练快

8. 决策树分裂准则
--------------------------------------------------------------------------------
  信息熵：H(S) = -Σ p_i·log₂(p_i)

  信息增益：Gain = H(parent) - Σ (|S_v|/|S|)·H(S_v)
  → 选择使信息增益最大的特征分裂（ID3算法）

  基尼系数：Gini = 1 - Σ p_i²
  → CART决策树默认用基尼系数

  优点：可解释，无需特征标准化，能处理非线性

9. 随机森林 vs XGBoost 对比
--------------------------------------------------------------------------------
  随机森林：
    - Bagging（独立训练多棵树，投票/平均）
    - 每棵树用随机样本 + 随机特征子集
    - 树之间相互独立，可并行
    - 抗过拟合能力强

  XGBoost：
    - Boosting（串行训练，每棵树的目的是拟合残差）
    - 加入了二阶导数信息、正则化项
    - 对缺失值自动处理
    - 精度通常高于随机森林

四、神经网络基础（★ 高频）
================================================================================

10. 激活函数对比
--------------------------------------------------------------------------------
  Sigmoid：  σ = 1/(1+e⁻ˣ)      输出(0,1)，易梯度消失，多用于二分类输出层
  Tanh：     tanh(x)           输出(-1,1)，零中心，仍易梯度消失
  ReLU：     max(0, x)         输出[0,∞)，不饱和，计算快，死亡神经元问题
  LeakyReLU：max(ax, x)        a≈0.01，死亡神经元少
  GeLU：     x·Φ(x)            Transformer标配（如BERT）

  梯度消失：sigmoid/tanh 在|x|大时梯度→0 → 深层网络前面层不更新
  解决：ReLU / BatchNorm / 残差连接 / 预训练

11. BatchNorm ★★★★★
--------------------------------------------------------------------------------
  公式：y = γ·(x - μ)/√(σ²+ε) + β

  训练时：μ, σ 从当前 batch 计算
  推理时：用全局移动平均（running mean/var）

  作用：
    ① 缓解Internal Covariate Shift（ICS）
    ② 使梯度更稳定，允许更大学习率
    ③ 本身有轻微正则化效果（依赖batch统计量）

  注意：batch size 过小时统计不稳定，效果差（≈1时基本失效）

12. Dropout
--------------------------------------------------------------------------------
  训练：随机以概率 p 丢弃神经元（前向时置0，反向时不传梯度）
  推理：保留全部神经元，所有权重乘以(1-p)（或训练时用inverted dropout）

  本质：每次训练一个子网络，推理时平均所有子网络

  作用：打破神经元间的协同依赖 → 正则化，防止过拟合

13. 前向传播与反向传播
--------------------------------------------------------------------------------
  全连接层：y = Wx + b
    前向：计算 y
    反向：∂L/∂W = δ·xᵀ，∂L/∂x = Wᵀδ

  计算图：从后往前逐层求导，用链式法则

五、卷积神经网络 CNN（★ 图像方向必考）
================================================================================

14. 卷积层参数量
--------------------------------------------------------------------------------
  参数量 = (K_h × K_w × C_in + 1) × C_out
    K = 卷积核尺寸，C = 通道数，+1 = 偏置

  例：输入32×32×3，卷积核3×3，64个输出通道
    params = (3×3×3 + 1) × 64 = 1792

  输出尺寸：H_out = (H_in + 2×pad - K)/stride + 1

15. 1×1 卷积的作用
--------------------------------------------------------------------------------
  ① 通道维度的线性组合（跨通道信息融合）
  ② 升维/降维（如 ResNet 中 BottleNeck：1×1降维→3×3→1×1升维）
  ③ 减少计算量（降维后3×3卷积计算量大幅下降）
  ④ 等效为全连接层在通道维度上操作

16. 经典网络结构
--------------------------------------------------------------------------------
  VGG-16：13层conv(3×3) + 3层FC，结构简单，参数量大（138M）
  ResNet：残差连接 y = F(x) + x，允许更深网络
    解决：深层网络退化（非过拟合，而是训练误差也变高）
    原理：恒等映射使梯度直达深层，网络至少不退化
  MobileNet：深度可分离卷积 + 倒残差结构，轻量高效
  DenseNet：每层与前面所有层密集连接，特征复用

17. 池化层
--------------------------------------------------------------------------------
  MaxPooling：取最大值，抗噪声，保留纹理特征
  AvgPooling：取均值，平滑，保留背景信息
  GlobalPooling：把H×W压成1×1，消除空间依赖

  反向传播：MaxPooling 反向时只传最大值位置的梯度

六、Transformer 与 Attention（★ 最高频之一）
================================================================================

18. Self-Attention 计算流程 ★★★★★
--------------------------------------------------------------------------------
  Input: Q = XW_Q,  K = XW_K,  V = XW_V

  Step 1: 打分
    scores = QKᵀ / √d_k

  Step 2: 归一化
    attn_weights = softmax(scores, axis=-1)

  Step 3: 加权求和
    output = attn_weights @ V

  √d_k 的作用：点积随 d_k 增大会变大 → softmax 梯度趋近0，需要缩放

  多头注意力：
    MultiHead = Concat(head₁,...,head_h) · W_O
    head_i = Attention(QW_i^Q, KW_i^K, VW_i^V)
    Transformer: h=8, d_model=512, d_k = d_v = d_model/h = 64

19. Transformer 架构要点
--------------------------------------------------------------------------------
  Encoder：6层，每层 = Multi-Head Self-Attention + FFN + 残差 + LayerNorm
  Decoder：6层，每层 = Masked MHA + Cross-Attention(看Encoder) + FFN + 残差 + LN

  LayerNorm：μ=0, σ=1，再γ缩放+β平移
  FFN：两层全连接，ReLU/GeLU激活
    FFN(x) = max(0, xW₁+b₁)W₂+b₂，等效kernel size=1的卷积

  位置编码：使模型感知token位置
    PE(pos, 2i)   = sin(pos / 10000^(2i/d))
    PE(pos, 2i+1) = cos(pos / 10000^(2i/d))

20. Attention 计算复杂度
--------------------------------------------------------------------------------
  Self-Attention：O(seq_len² · d_model)
  序列越长，注意力计算量爆炸式增长 → 长序列瓶颈

  优化方向：Sparse Attention / Linear Attention / Flash Attention

七、模型优化与训练（★ 常见概念题）
================================================================================

21. 梯度下降族（SGD / Momentum / Adam）
--------------------------------------------------------------------------------
  SGD: w = w - lr·g_t

  Momentum（动量）：
    v_t = β·v_{t-1} + (1-β)·g_t
    w_{t+1} = w_t - lr·v_t
    作用：加速收敛，冲破局部最优，减小振荡

  Adam（自适应学习率，当前默认）：
    m_t = β₁·m_{t-1} + (1-β₁)·g_t              一阶矩估计
    v_t = β₂·v_{t-1} + (1-β₂)·g_t²            二阶矩估计
    m̂_t = m_t / (1-β₁ᵗ)                       偏置校正
    v̂_t = v_t / (1-β₂ᵗ)
    w_{t+1} = w_t - lr·m̂_t / (√v̂_t + ε)

    常用参数：lr=0.001, β₁=0.9, β₂=0.999, ε=1e-8

22. 学习率策略
--------------------------------------------------------------------------------
  固定：全程lr不变
  阶梯衰减：每N个epoch乘以γ
  余弦退火：lr = lr_min + 0.5·(lr_max-lr_min)·(1+cos(π·t/T))
  Warmup：初期逐步增大学习率，防止早期震荡

  作用：学习率过大→不收敛；过小→收敛慢

23. 梯度消失与爆炸
--------------------------------------------------------------------------------
  梯度消失：链式法则连乘导致梯度→0，深层几乎不更新
    症状：loss下降极慢，前几层参数几乎不变
    解决：ReLU / BatchNorm / 残差连接 / 预训练 / LSTM门控

  梯度爆炸：梯度→∞，参数剧烈更新，loss NaN
    症状：loss突然变NaN，权重变inf
    解决：梯度裁剪（clip_grad_norm）、权重正则化、降低学习率

八、模型压缩与部署（★ 常见概念题）
================================================================================

24. 量化（Quantization）
--------------------------------------------------------------------------------
  FP32 → INT8：参数量×1/4，显存×1/4，计算更快

  对称量化（最常见）：
    scale = |X|_max / 127
    Q = round(X / scale)
    X̂ = Q × scale

  量化误差来源：
    ① 舍入误差（round操作）
    ② 溢出（超出[-127,127]截断）
    → per-channel 量化可减少误差（每个channel独立scale）

  QAT（Quantization Aware Training）：
    训练时模拟量化效果，精度比后训练量化更高

25. 剪枝（Pruning）
--------------------------------------------------------------------------------
  权重剪枝：将小于阈值的权重置0
  结构化剪枝：整组（channel/filter）剪枝，硬件友好
  训练中剪枝：迭代剪枝（train→prune→fine-tune循环）

26. 知识蒸馏（Knowledge Distillation）
--------------------------------------------------------------------------------
  教师模型（复杂）→ 软标签（logits）→ 学生模型（轻量）学习
  损失 = α·CE(学生硬标签) + (1-α)·KL(学生软概率, 教师软概率)

  作用：把大模型知识压缩到小模型，用于部署

九、显存与计算量估算（★ 常考应用题）
================================================================================

27. 显存组成
--------------------------------------------------------------------------------
  显存 = 模型参数 + 梯度 + 优化器状态 + 激活值 + 中间缓存

  全连接层：params = (d_in + 1) × d_out
  梯度：与参数量相同
  优化器状态（Adam）：2 × params

  激活值：
    Conv: B × H_out × W_out × C_out × 2bytes(FP16)
    Attention: B × N × N × h × 2bytes（N=seq_len, h=heads）

28. 显存优化策略
--------------------------------------------------------------------------------
  ① 梯度累积：batch大→拆成小步，累加梯度后更新 → 等效大batch
  ② 混合精度训练：前反向FP16，参数+优化器FP32 → 省显存+加速
  ③ Gradient Checkpointing：用计算换显存（重新计算激活值）
  ④ 模型并行/TP/PP：切分模型到多卡，流水线减少空泡
  ⑤ CPU Offload：把部分参数/优化器状态卸载到CPU内存

29. FLOPs / MACs 估算
--------------------------------------------------------------------------------
  1 MAC ≈ 2 FLOPs

  全连接层：MACs ≈ d_in × d_out
  卷积层：MACs = H_out × W_out × C_in × C_out × K_h × K_w
  Self-Attention：MACs ≈ 2 × N² × d（N=seq_len）

十、其他高频概念（★ 选择填空常考）
================================================================================

30. 数据增强（Data Augmentation）
--------------------------------------------------------------------------------
  图像：随机翻转 / 旋转 / 裁剪 / 颜色抖动 / Mixup / CutMix
  文本：同义词替换 / 回译 / 随机删除 / 数据增强EDA
  作用：扩充训练集多样性，提升泛化能力，正则化

31. 交叉验证（Cross Validation）
--------------------------------------------------------------------------------
  K-Fold CV：均分K份，轮流用1份测试，K-1份训练
  LOOCV：每份1个样本，适合小数据集
  用途：评估模型泛化能力，超参数选择

32. 类别不平衡处理
--------------------------------------------------------------------------------
  ① 过采样：SMOTE生成少数类样本
  ② 欠采样：减少多数类样本
  ③ 类别权重：损失函数中给少数类更大权重
  ④ 阈值调整：降低判为正例的阈值，提高召回
  ⑤ 评估指标：用AUC/F1/Recall代替Accuracy

33. RAG 系统（检索增强生成）
--------------------------------------------------------------------------------
  Retrieval Augmented Generation：
    检索（Retrieval）→ 获取相关文档片段
    增强（Augment）→ 将检索结果加入Prompt
    生成（Generate）→ LLM基于增强后的Prompt生成答案

  核心组件：Embedding模型 + 向量数据库 + 重排序模型
  作用：解决LLM知识过时/幻觉问题，接入最新知识

34. 词嵌入（Word2Vec / Embedding）
--------------------------------------------------------------------------------
  Word2Vec：CBOW（上下文→中心词）vs Skip-gram（中心词→上下文）
  负采样：减少softmax分母计算量
  效果：语义相近的词向量距离近
